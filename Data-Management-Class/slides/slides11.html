Final Project
 for either DMP or data deposit
  1. An abstract with a clear description of purpose
  2. Clear use of ‘data’ language (data types, data formats, data stages)
  3. Clear use of ‘file system’ language (file names, folder structures)
  4. Clear description of process steps from ‘raw’ data to ‘published’ data
  5. A assessment of the data accuracy (known errors, problems with process)
  6. Clear statement on ownership of data and how it is to be shared (license)
  7. Description and rationalization of metadata standard used (or not)
  8. A description of past and future versions of the data (or not)
Please send a one sentence description of your project to:
tnorris@miami.edu and aclark@rsmas.miami.edu by Monday April 10th.
